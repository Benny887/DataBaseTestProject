1. Для создания приложения использовались jdk1.8.0.311,Maven,PostgreSQL 13, IDE IntelliJ IDEA, кодировка UTF-8
2. Для настройки соединения с БД исползуется файл расположенный в resources\db.properties
3. Используемые зависимости расположены в pom.xml
4. Предварительно необходимо убедиться в запуске СУБД PostgreSQL,а также в создании БД указанной в файле db.properties
5. Приложение поддерживает операции в соответствии с ТЗ
6. Запуск приложения осуществляется с командной строки,с помощью jar-файла расположенного в папке UserFiles
6. При старте приложения запускается класс для построения необходимых для работы таблиц,после чего, при первом старте, программа
   приостановится и поступит предложение о заполнении БД. Данные для заполнения таблиц (дамп), находятся в файле storeDump, расположенном
   в папке UserFiles, который необходимо открыть из СУБД PostgreSQL и запустить. После этого нажать Enter в консоли,
   программа продолжит работу.
7. Особое внимание обратите на выбор нужной кодировки файла вывода, для корректного отобажения результатов запросов.
8. Набор данных для тестирования расположен в папке UserFiles\TestFiles\SearchFuncTest - для проверки операции "search",
   UserFiles\TestFiles\StatFunctionTest - для проверки операции "stat".
